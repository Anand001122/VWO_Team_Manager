# VWO Team Manager Dashboard

A Streamlit-based application to manage team performance and track development metrics.

## Features
- **Team Overview:** View member roles, status, and performance scores.
- **Data Integration:** Built with Pandas for data handling.
- **Modular Design:** Separate modules for Git and Jira analysis.

## How to Run Locally
1. Clone the repository.
2. Run `pip install -r requirements.txt`.
3. Execute `streamlit run dashboard.py`.

## Tech Stack
- Python
- Streamlit
- Pandas
